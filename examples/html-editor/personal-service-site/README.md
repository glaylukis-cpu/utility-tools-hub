# Studio Note - HTML Editor作成例

架空の個人業務サポートサービスを紹介する、1ページサイトのサンプルです。

## ファイル

- `studio-note.uth-html-editor.json`: Utility Tools Hub HTML Editorで開いて再編集するプロジェクト
- `index.html`: ブラウザや静的ホスティングで確認できる掲載用HTML

## HTML Editorで開く

1. Utility Tools HubでHTML Editorを開きます。
2. Projectの「Open Project...」を選びます。
3. `studio-note.uth-html-editor.json`を開きます。
4. 文言、色、余白、幅を編集し、「Copy HTML」でHTMLを取得します。

この例は無料プレビューで使える単一ページと基本ブロックだけで構成しています。ボタンのリンクは安全なプレースホルダーとして`#`にしているため、公開前に実際の問い合わせ先へ変更してください。外部フォント、外部画像、JavaScriptは使用していません。
